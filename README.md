## Database team pj Phase 3

- JDBC programming
- CLI interface

## How to run

1. Get this repository

```shell
git clone https://github.com/DBProjectTeam15/Phase3
```

- For IntelliJ users

Simply, you just open the project folder as root. And you can run `Main` function by opening this project.

- For Eclipse users 

Follow the instructions below in Eclipse IDE (if you want to run this in Eclipse.)


```shell
1. File -> Import

2. Search `Gradle` and `Existing Gradle Project`

3. Find project root directory
```

Finally, Right click `Project` and navigate to `Gradle`, and Press `Refresh...`
